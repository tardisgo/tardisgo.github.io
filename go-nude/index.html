Nudity Detection: A Go library called from Haxe

The go-nude nudity detection libray takes a jpeg, gif or png image and determines if it contains nudity. Try it below by uploading a thumbnail image, which will be processed in your browser.

The library has been translated from Go into Haxe using TARDIS Go. The Haxe compiler then translated this program to call the library, and the generated library itself, into JS to enable it to run in this page.

The translation process is currently very imature and produces slow code. As a result, please only test with small thumbnail-size images. Finally, be aware that the library does not always give the result you might expect!
